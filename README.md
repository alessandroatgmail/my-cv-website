# my-cv-website

A personal CV website for Alessandro Silvestrin, publicly hosted online. The site features a bilingual interface (EN/IT), a light/dark theme toggle, a live visit counter, an AI-powered chatbot, and visual skill charts.

## Features

- **Bilingual (EN/IT)** — full content translation switching at runtime
- **Light/Dark theme** — toggle stored in user preferences
- **Live visit counter** — page view count fetched from a Cloud Function backed by Firestore
- **RAG Chatbot** — ask questions about Alessandro's background; answers are generated by Gemini using BM25 retrieval over a knowledge base stored in Cloud Storage
- **Skill charts** — radar and donut charts (Python-generated PNGs) embedded in the page

## Architecture

```
Browser (HTML/CSS/JS)
    │
    ├── GET  /counter      → Cloud Function (Python) → Firestore
    └── POST /rag-chatbot  → Cloud Function (Python) → Cloud Storage (index.json) + Vertex AI (Gemini)
```

The knowledge base is pre-built by a separate **rag-indexer** Cloud Function that chunks documents and writes `index.json` to Cloud Storage.

## Technologies

| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Backend | Python 3, Google Cloud Functions Framework |
| AI / LLM | Vertex AI — Gemini 2.0 Flash |
| Retrieval | BM25 (custom implementation) |
| Storage | Google Cloud Storage (knowledge base index) |
| Database | Google Cloud Firestore (visit counter) |
| Cloud | Google Cloud Platform (GCP) |
